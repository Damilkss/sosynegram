# Sosynegram 📸✨

**Sosynegram** — это современная социальная сеть, вдохновленная Instagram. Платформа позволяет пользователям делиться моментами своей жизни, общаться и находить новых друзей в уютном и безопасном сообществе.

## ✨ Возможности

- 📝 **Регистрация и авторизация** — безопасный вход с хешированием паролей
- 👤 **Профили пользователей** — аватарки, личная информация, лента активности
- 📷 **Публикация постов** — делитесь фото и мыслями с подписчиками
- 💬 **Комментарии** — обсуждайте посты с друзьями
- 🔔 **Уведомления** — будьте в курсе лайков, комментариев и новых подписок
- 👥 **Подписки и подписчики** — следите за любимыми авторами
- 🏷️ **Упоминания** — отмечайте друзей в комментариях
- 🖼️ **Загрузка медиа** — поддержка изображений и видео
